# AutoTailor

AutoTailor — это инструмент для автоматической предварительной обработки сканированных документов и верификации текста через OCR. Он предназначен для очистки, выравнивания, обрезки и проверки сканированных чертежей, планов и общих документов.

---

## Возможности

* **Интеллектуальный автоповорот (выравнивание):** Автоматически определяет и корректирует ориентацию изображения (0, 90, 180, 270 градусов) на основе анализа ключевых слов через Tesseract OCR.
* **Контентно-зависимая обрезка полей:** Находит фактические границы документа с помощью анализа связанных компонентов, отфильтровывает тени и посторонние шумы от стола по краям, выравнивая и заливая поля чистым белым цветом.
* **Адаптивная очистка фона (3 режима):**
  * `gray` (Оттенки серого - по умолчанию): Нормализует фон методом гауссовой нормализации и попиксельного нахождения минимума цветовых каналов. Это сохраняет все цветные элементы (синие печати, подписи, оранжевые заливки) как контрастные темно-серые, делая сам лист идеально белым.
  * `bw` (Черно-белый): Бинаризует скан с использованием адаптивного порога для получения высокого контраста текста.
  * `color` (Цветной): Выравнивает яркость и убирает тени по всем цветовым каналам для чистого цветного скана.
* **Двуязычный интерфейс и отчеты:** Полная поддержка логов консоли и генерируемых отчетов на русском и английском языках (настраивается на лету).
* **Конфигурация через JSON:** Возможность настройки списков ключевых слов для проверок, языков OCR и директорий в удобном формате.

## Установка

### 1. Установка Tesseract OCR

Для работы AutoTailor необходимо установить Tesseract OCR в вашей системе.

* **Windows:**
  Скачайте и запустите установщик от [UB-Mannheim Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki).
  *(Не забудьте выбрать украинский и русский языковые пакеты в процессе установки, если они вам нужны)*.
* **macOS:**
  ```bash
  brew install tesseract tesseract-lang
  ```
* **Linux (Ubuntu/Debian):**
  ```bash
  sudo apt update
  sudo apt install tesseract-ocr tesseract-ocr-ukr tesseract-ocr-rus
  ```

### 2. Установка AutoTailor

Клонируйте данный репозиторий и установите его с помощью `pip`:

```bash
git clone https://github.com/yourusername/AutoTailor.git
cd AutoTailor
pip install .
```

## Настройка (`config.json`)

Проект использует файл `config.json` в корневой директории. Вы можете изменить его для настройки поведения:

```json
{
  "mode": "gray",
  "ocr_language": "ukr+eng",
  "ocr_checklist": [
    "СХЕМАТИЧНИЙ",
    "ПЛАН",
    "Кривий",
    "Ріг"
  ],
  "rotation_keywords": [
    "план",
    "поверх",
    "область",
    "експлікація"
  ],
  "report_language": "ru",
  "tessdata_dir": null
}
```

### Параметры конфигурации:
* `mode`: Режим обработки по умолчанию (`gray`, `bw` или `color`).
* `ocr_language`: Языки Tesseract для OCR (например, `ukr+eng` или `eng+rus`).
* `ocr_checklist`: Ключевые слова, которые должен содержать документ для успешной верификации.
* `rotation_keywords`: Слова, используемые алгоритмом автоповорота для определения правильной ориентации страницы.
* `report_language`: Язык вывода в консоль и генерируемых текстовых отчетов (`ru` для русского, `en` для английского).
* `tessdata_dir`: Абсолютный путь к папке `tessdata` Tesseract. Если указано `null`, AutoTailor автоматически найдет ее в локальных или системных путях.

## Использование

### Интерфейс командной строки (CLI)

После установки через pip вы можете вызывать утилиту напрямую:

```bash
# Обработка одного файла
autotailor path/to/input.jpg path/to/output_dir/ 0 gray

# Обработка всех файлов в папке inbox
autotailor --inbox
```

Также вы можете запускать его как модуль Python из корня проекта:

```bash
# Пакетная обработка файлов из папки inbox
python -m autotailor.cli --inbox
```

### Батники (Windows)
Вы можете дважды кликнуть на скрипты в корне проекта для смены режимов и запуска:
* `Process_Inbox.bat`: Запуск автоматической пакетной обработки файлов в папке `inbox/`.
* `Set_Mode_Gray.bat`: Установить режим очистки фона в оттенки серого.
* `Set_Mode_BW.bat`: Установить режим очистки фона в черно-белый (высокий контраст).
* `Set_Mode_Color.bat`: Установить режим очистки фона в цветной.
