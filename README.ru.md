# AutoTailor

AutoTailor — это автоматическая программа для очистки, поворота и обрезки отсканированных чертежей, планов и документов.

Программа автоматически удаляет серый фон сканера (делая бумагу идеально белой), переворачивает страницы текстом вверх, обрезает лишние края и распознает текст.

### 🚀 Умная автонастройка (Zero-Setup)
В AutoTailor встроен интеллектуальный режим автоопределения. Когда вы запускаете обработку папки с файлами, программа автоматически сканирует первый лист, определяет язык и тип документов (русский, украинский или английский) и сама загружает подходящие словари для поиска, правила переворота страниц и язык отчетов. **Вам не нужно настраивать программу вручную — она всё сделает сама.**

## Быстрый старт (для пользователей Windows)

Вам не нужно уметь программировать или настраивать окружение вручную. Вся настройка AutoTailor происходит автоматически.

### 1. Первоначальная настройка (выполняется один раз)
1. Дважды кликните по файлу **`setup_windows.bat`** в папке программы.
2. Скрипт сам автоматически установит **Python**, систему распознавания **Tesseract-OCR** и все необходимые компоненты.
3. Дождитесь окончания установки. Если Windows запросит разрешение на установку программ, подтвердите его.

### 2. Обработка документов
1. Скопируйте ваши отсканированные файлы (изображения JPG, PNG, BMP) в папку **`inbox`**.
2. Дважды кликните по файлу **`Process_Inbox.bat`**.
3. Программа сама очистит фон, выровняет ориентацию страниц и сохранит результаты.
4. Откройте папку **`out`** — там будут лежать очищенные файлы PNG, собранные PDF-документы и текстовые отчеты с распознанным текстом.

### 3. Смена режимов очистки
Вы можете изменить то, как программа очищает сканы, просто запустив один из файлов настройки в папке проекта:
* **`Set_Mode_Gray.bat`** (По умолчанию): Очищает фон до белого, но сохраняет цветные печати, подписи и пометки.
* **`Set_Mode_BW.bat`**: Переводит скан в чистый черно-белый формат с высоким контрастом (подходит для текстовых документов).
* **`Set_Mode_Color.bat`**: Убирает тени и серый налет, полностью сохраняя исходные цвета.

## Настройка параметров (`config.json`)

Вы можете открыть файл `config.json` в любом текстовом редакторе (например, в Блокноте), чтобы настроить ключевые слова под ваши типы документов.

Пример настроек с общими параметрами:
```json
{
  "mode": "gray",
  "ocr_language": "rus+eng",
  "ocr_checklist": [
    "СХЕМАТИЧЕСКИЙ",
    "ПЛАН",
    "ЭКСПЛИКАЦИЯ"
  ],
  "rotation_keywords": [
    "план",
    "этаж",
    "область",
    "улица",
    "экспликация",
    "здание",
    "схема"
  ],
  "report_language": "ru",
  "tessdata_dir": null
}
```

* `ocr_checklist`: Ключевые слова, которые программа ищет на чертеже/документе для проверки его типа (если слово найдено, документ считается верным).
* `rotation_keywords`: Слова, по которым программа понимает, как перевернуть лист (ищет слова типа "план" или "улица" и поворачивает лист так, чтобы эти слова читались горизонтально).
* `report_language`: Язык логов консоли и текстовых отчетов (`ru` — русский, `en` — английский).

## Установка для разработчиков (Advanced)

Если вы разработчик, вы можете установить библиотеку через pip:

```bash
pip install .
```

И использовать через консоль:
```bash
# Обработать один файл
autotailor путь/к/файлу.jpg путь/к/выходной_папке/

# Обработать всю папку inbox
autotailor --inbox
```
